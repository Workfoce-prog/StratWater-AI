
# StratWater AI Demo App

This Streamlit app provides a demo dashboard for a Mali-first West Africa pilot.

## Features
- Village overview and KPIs
- Daily water-use trends
- AI-style reliability, maintenance-risk, stress, and sustainability scores
- Maintenance log
- Financial summary
- Map view
- Optional CSV upload for replacement village data

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Sample files
- `data/villages.csv`
- `data/daily_usage.csv`
- `data/maintenance_log.csv`

## Note
All values are illustrative demo values for pitch and pilot-planning use.
